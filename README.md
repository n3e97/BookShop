## BookShop
此为软件工程训练的大作业，为方便团队协作，故而建了一个github项目.没钱就只能弄成公开项目了。

##简要描述
>网络应用日益生活化，消费者越来越多的日常需求都通过利用网络工具得到了很好的实现。网上购物成为
众多消费者生活中不可或缺的一部分。这一发展趋势给线下商家带来了巨大的挑战。为了得到更好的发展，
线下商家也开始寻求互联网带来的信息便利，通过充分利用这一平台，争取更广泛的客户群。

>书店行业基于自身商品种类多、查找困难等突出特点，更需要一个信息展示的平台向顾客展示其商品信息，
摆脱实体店铺的束缚，使购书过程更加便捷。
